# 手动测试清单

在支持 Markdown 的编辑器（或 GitHub / GitLab）中，将 `- [ ]` 改为 `- [x]` 即可标记为已完成。

---

## Chat（前端 Chat 页 → `POST /api/chat`）

- [ ] **无命中 / 与文档无关**：在已配置 **`VOLCENGINE_API_KEY`** 与 **`LLM_MODEL`** 时，应得到**火山 LLM 返回的回答**（非本地固定模板），响应里 **`sources` 为空**，`confidence` / `context_used` 为 0。
- [ ] **与文档相关且能命中**：仍应是**带引用片段的 RAG**（`sources` 非空，回答依据文档）。
- [ ] **无命中且未配 Key 或未配 `LLM_MODEL`**：应得到**仅说明需配置环境变量**的短文案，**不调用**大模型。

## `POST /api/query`（Swagger `/docs` 或 curl）

- [ ] **无命中**：行为与 Chat **一致**——已配 Key/模型时 **`answer` 为 LLM 生成**（`sources` 为空等）；未配置时同上为配置提示。
